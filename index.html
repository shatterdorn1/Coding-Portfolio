HTML Cheatsheets

HTML Tag Name Description Example
<HTML> HTML Hyper Text Markup Language Used at the start of an HTML document to determine the documents language
<head> Head This is a container for all the head elements. Title for the document, scripts, styles, meta and more.
<title> Title This is the title of the webpage when you mouse over a tab or look at the top of the browser.
<body> Body Only content inside the body tags will be displayed to the screen.
<div> division Div is the generic container for flow content. It is nonspecific. <div id="section 1">
<h1> h1 H1 is a heading tag, used with the other 5 headings

This is a Heading
<p> Paragraph P is used for displaying plain text.

This is normal text.
<em> Emphasis This element makes text appear in Italics for emphasis.

This text is in Italics.
<strong> Strong This element makes text appear bold.

This text is Bold.
<br> Line Break This is a line break
<ul> Unordered List This element will put items in a list with bullet points.
<ol> Ordered List This element creates a numbered list.
<li> List Item This is an element that goes inside both Unordered and Ordered Lists.
<img> Image This will place an image in the document. It MUST have a source. <img src="insert URL" width="200" height="200" alt="description">
<video> Video This will place a video in the document. It MUST have a source. <video width="320" height="320" controls><sourc src="movie.mp4" type="video/mp4"></video>
<a> Anchor This is for creating a hyperlink. <a href="url" target_"blank">Text</>a>
<!-- /--> Comment This will add a grey comment that only shows up when looking at the code.
